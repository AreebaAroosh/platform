Core

@ngez/core is a library for Angular applications, which consists of minimalistic components which are fully customizable and easy to override with new styles. Useful if you don't want to be constrained by Material or Bootstrap's theme. You take care of the view, we'll take care of the functionality.

Everything's tested on Chrome, Firefox and IE11.
